Set User Role

Grant admin or mentor privileges to a user

Security Warning Admin: Can create/modify any course and access admin-only features.
Mentor: Can create courses and edit content of their own courses only (cannot change title, price, category, etc.).
How to use:
1. Enter the email address of the user
2. Select the role (Admin, Mentor, or User)
3. Click "Set Role"

Role Differences:
• Admin: Can create/modify any course, change all fields
• Mentor: Can create courses and edit content (description, lessons, resources, assignments) of their own courses only
• User: Can only enroll in courses

Note: You must be logged in as admin to use this tool.
